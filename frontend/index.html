パスワード不要。指紋・顔・デバイスPINで安全にサインイン。フィッシング耐性があり、パスワード漏洩リスクがありません。

AES-GCM-256暗号化

TODOの内容はあなたのデバイス上でAES-GCM-256で暗号化。サーバーには暗号文のみが保存されます。

🚫

ゼロナレッジ設計

サーバーは暗号化データしか持ちません。管理者・開発者であってもTODOの内容を読むことはできません。

プライバシーポリシー

最終更新: 2025年4月

収集するデータ

  • ユーザー名 — 認証識別子として保存（平文）
  • WebAuthnクレデンシャル — パスキーの公開鍵情報（秘密鍵はデバイス上のみ）
  • 暗号化TODOデータ — AES-GCM-256で暗号化された内容のみ。サーバーは復号不可

収集しないデータ

  • TODOの平文コンテンツ（サーバーは暗号文しか持ちません）
  • IPアドレス・ユーザー行動ログ・アクセス解析
  • 第三者トラッキング・広告

暗号化について

暗号化鍵はWebAuthn PRF拡張から導出され、あなたのデバイスのメモリ上にのみ存在します。 サーバーに送信されることはありません。ページを閉じると鍵は消失し、 次回アクセス時にパスキー認証で再生成されます。

データの削除

アカウントの削除を希望する場合は、 GitHub Issues よりお問い合わせください。サーバー上のデータ（暗号化済みTODO・クレデンシャル）をすべて削除します。

第三者への提供

収集したデータを第三者に提供・販売することはありません。
